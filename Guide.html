02 Client Logos

Currently the client grid shows text names. To show logos instead, replace the text inside each .client-card with an <img> tag.

Current (text only):

<div class="client-card">Boult Audio</div>

Replace with (logo image):

<div class="client-card"> <img src="images/logos/boult-audio.png" alt="Boult Audio"> </div>

The CSS already handles logo styling:

  • Logos auto-resize to max 120px wide, 40px tall
  • They appear desaturated (grayscale) by default
  • On hover, they become full color
  • The alt text is used for accessibility
Logo specs: PNG with transparent background. Width: 200-400px. Keep files under 50KB each.

03 Latest + Upcoming Cards

These are the big horizontal-scroll cards in the white section. Each card currently uses a gradient background. To add a real image:

Current (gradient placeholder):

<div class="latest-card-thumb"> <div class="thumb-bg" style="background:linear-gradient(...)"></div> <div class="thumb-overlay">...</div> </div>

Add image (keep gradient as fallback):

<div class="latest-card-thumb"> <div class="thumb-bg" style="background:linear-gradient(...)"></div> <img src="images/latest/anjumkhan-latest.jpg" alt="Anjum Khan"> <div class="thumb-overlay">...</div> </div>

The image sits on top of the gradient (z-index:1), so the gradient only shows while the image loads. The overlay text stays on top (z-index:2) for readability.

Image specs: 4:5 aspect ratio (800×1000px recommended). JPG, optimized to under 200KB.

04 Social Media Section Cards

The "SOCIAL MEDIA" horizontal scroll section on the homepage. Same principle — add an <img> inside .cat-card-thumb:

Add image to a category card:

<div class="cat-card-thumb"> <div class="cat-thumb-bg" style="background:linear-gradient(...)"></div> <img src="images/social-media/anjumkhan-cover.jpg" alt="Anjum Khan"> <div class="cat-thumb-content">...</div> </div>

The overlay text (brand name, badge) stays visible on top of the image.

Image specs: 3:4 aspect ratio (600×800px recommended). JPG. Use your best design mockup or portfolio screenshot for each brand.

05 Meta Ads Section Cards

Exact same process as Social Media above. Find the "META ADS" .category-section block in index.html and add <img> tags inside each .cat-card-thumb.

<!-- Inside the Meta Ads section --> <div class="cat-card-thumb"> <div class="cat-thumb-bg" style="background:..."></div> <img src="images/ads/butternut-ads-cover.jpg" alt="Butternut Co. Ads"> <div class="cat-thumb-content">...</div> </div>

06 Category Subpages (social-media.html, ads.html, etc.)

Each category page (social-media.html, ads.html, listings.html, banners.html) uses .brand-card components with gradient backgrounds. To add real images:

Current brand card:

<div class="card-visual"> <div class="gradient-bg" style="background:linear-gradient(...)"></div> <span class="brand-name">Anjum Khan</span> </div>

With real image:

<div class="card-visual"> <div class="gradient-bg" style="background:linear-gradient(...)"></div> <img src="images/social-media/anjumkhan-cover.jpg" alt="Anjum Khan Social Media" style="position:absolute;inset:0;width:100%;height:100%;object-fit:cover"> <span class="brand-name">Anjum Khan</span> </div>
Important: The subpage HTML files (social-media.html, ads.html, etc.) don't have the img CSS built in yet. You need to add the inline style style="position:absolute;inset:0;width:100%;height:100%;object-fit:cover" on the img tag, or add this CSS rule to their <style> block:

.card-visual img { position:absolute; inset:0; width:100%; height:100%; object-fit:cover; }

07 Adding a New Category

To add a new category (e.g., "YouTube Thumbnails") to the homepage:

Step 1: Add a new section in index.html

Copy an existing .category-section block and paste it. Change the heading, scroll ID, and card content:

<!-- NEW CATEGORY: YouTube Thumbnails --> <div class="category-section"> <div class="category-section-header"> <h2>YouTube Thumbnails</h2> <!-- Copy scroll-nav buttons, change IDs --> </div> <div class="category-scroll" id="ytScroll"> <!-- Add .cat-card items here --> </div> </div>

Step 2: Add a work card in the "Selected Work" grid

<a href="youtube.html" class="work-card reveal"> <div class="card-top"> <span class="card-number">05</span> <!-- Copy arrow div from existing card --> </div> <div class="card-content"> <div class="card-title">YouTube Thumbnails</div> <div class="card-desc">Click-worthy thumbnails that boost video CTR.</div> </div> </a>

Step 3: Create the subpage

Copy social-media.html, rename to youtube.html, and update the header + brand cards inside it.

09 Deploying Changes

  • If hosted on GitHub Pages: commit and push changes to your repository
  • If hosted on Netlify/Vercel: drag and drop the entire folder to redeploy
  • If using a custom hosting (Hostinger, etc.): upload changed files via FTP or file manager
  • Always test on both desktop and mobile after deploying
  • Clear browser cache (Ctrl+Shift+R) to see latest changes
Quick checklist before publishing: All images load correctly? All links work? Mobile looks good? Text content is accurate? Loading speed is acceptable?
